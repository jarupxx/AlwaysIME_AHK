# AlwaysIME_AHK

## 概要

IME自動オンにするソフト


## 主な機能

自動ON切り替え: アプリケーションを切り替えた際、通常はオフ（半角英数）に戻ってしまうIMEを、自動でオン（ひらがな）の状態にします。

個別アプリ設定: 特定のゲームやアプリなど、常にオフにしておきたいソフトを「除外リスト」に登録できます。

ウィンドウタイトル判定: アプリ名だけでなく、特定のウィンドウタイトルに基づいてIMEの挙動を制御することも可能です。 

## 動作環境

Windows 11
